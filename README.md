# ZootilityBelt

**ZootilityBelt** is a collection of practical web-based tools designed to help zoo staff, zookeepers, and animal management professionals handle daily tasks more easily and efficiently.

Live Demo: [https://zootilitybelt.github.io/Main-menu/](https://zootilitybelt.github.io/Main-menu/)

## 🛠️ Features / Tools Available

The application provides a central menu with access to the following specialized tools:

### 1. Ear Notch Generator
Generate ear notch patterns based on internal ID numbers for hoofstock identification. This helps quickly visualize and plan ear notching for new animals or identify existing ones.

### 2. Reptile Energy Calculator
Calculate precise energy requirements and plan feeding schedules for reptiles. This ensures optimal nutrition and metabolic health based on the species' specific needs.

### 3. Ferguson Zone Finder
Find the correct Ferguson Zone and UV (ultraviolet) lighting requirements for various reptile species, ensuring they receive the appropriate UVB exposure for synthesizing Vitamin D3.

### 4. Enrichment Randomizer
Create randomized weekly and daily enrichment schedules for zoo animals. This helps keep enrichment fresh, unpredictable, and mentally stimulating for the animals, while saving keepers time in planning.

### 5. QBA Scoring Tool
Qualitative Behaviour Assessment (QBA) tool for evaluating and recording animal emotional expression and overall welfare. 

## 🚀 Getting Started

To use the tools locally or contribute to the project:

1. Clone the repository:
   ```bash
   git clone https://github.com/ZootilityBelt/Main-menu.git
   ```
2. Open the directory:
   ```bash
   cd Main-menu
   ```
3. Open `index.html` in your preferred web browser. No complex build tools or servers are required to run the basic application.
